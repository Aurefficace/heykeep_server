# heykeep_server
## Lancer en local
Avoir PHP, symfony CLI et composer d'installés.  

La première fois :  
`composer install`  

Pour lancer en local sur : http://127.0.0.1:8000/  
`symfony server:start`  

Pour arrêter :  
`symfony server:stop`
